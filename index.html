What Is Closed-Source Software?

Not Publicly Available

The source code isn't publcily viewable or modifiable.

Exclusive Rights

The software owner can restrict use, inspection, and redistribution.

Trade Secrets

Source code is treated as trade secrets and are heavily guarded.

Examples of Closed-Source Software

What Is Open-Source Software?

Code Is Publicly Available

The source code is publicly viewable, modifiable, and enhanceable.

Community Driven

The direction and development of the project is driven by members of the community.

Meritocracy

The best ideas win and everyone has access to the same information.

Examples of Open-Source Software

Intra-Source Software

The integration and application of open-source best practices to closed-source projects to build effective community-driven software development projects.

Contributer Funnel

By: Mike McQuaid, maintainer of the Hombrew project for 7 years

The contributer funnel helps to visualize how a potential user might theoretically make their way to the bottom to become an active maintainer. The ultimate goal is to reduce friction between each stage of the contributer experience.

  • UsersUsers of the actual project
  • ContributorsSubmits issues and submits pull requests
  • MaintainersActively maintain the project

Documentation


  • Ensure it is easy for someone to actually use your project. Having a clear and friendly README.md will allow users to understand what the project is and what problem it solves.
  • Explain how to contribute to your project. A CONTRIBUTING.md file allows contributers to get up to speed to start contributing.
